# 정보보안기사 학습앱

## 프로젝트 개요
정보보안기사 시험 대비를 위한 Next.js 기반 학습 웹앱

## 기술 스택
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Playwright (문제 추출용)

## 주요 기능
1. **학습 카드** - 개념별 학습 자료
2. **퀴즈** - 기본 문제 풀이
3. **문제 은행** - 743문제 실전 연습
4. **AI 퀴즈** - Claude API로 문제 생성
5. **약점 관리** - 취약 분야 집중 학습

## 파일 구조
```
src/
├── app/
│   ├── page.tsx          # 메인 페이지
│   ├── layout.tsx        # 레이아웃
│   ├── globals.css       # 전역 스타일
│   └── api/
│       └── generate-quiz/route.ts  # AI 퀴즈 API
├── components/
│   ├── Diagrams.tsx      # 시각화 컴포넌트
│   └── Tables.tsx        # 테이블 컴포넌트
├── data/
│   ├── studyData.ts      # 학습 콘텐츠
│   └── questionBank.ts   # 문제 은행 (743문제)
└── scripts/
    ├── extract-questions.ts  # 문제 추출
    ├── process-bank.js       # 데이터 처리
    └── merge-banks.js        # 데이터 병합
```

## 문제 은행 현황
- 총 743문제 (중복 제거 후)
- 1과목: 시스템 보안 (111)
- 2과목: 네트워크 보안 (110)
- 3과목: 애플리케이션 보안 (84)
- 4과목: 정보보안 일반 (148)
- 5과목: 관리 및 법규 (141)
- 기타 (149)

## 개발 명령어
```bash
npm run dev          # 개발 서버 (포트 3000)
npm run build        # 프로덕션 빌드
```

## 문제 추출
```bash
npx ts-node src/scripts/extract-questions.ts "URL" [문제수]
node src/scripts/merge-banks.js
```

## 학습 진행 방식
CLI에서 직접 문제 풀이 및 개념 정리 가능
웹앱에서 시각적 학습 및 모의고사 진행

## 문제 풀이 가이드 (Claude 필독)

### 문제 풀이 후 반드시 해야 할 일

1. **맞았어도 물어보기**
   - "약점에 추가할까요?" 항상 확인
   - 헷갈렸거나 찍은 문제는 추가 권장

2. **틀렸으면 무조건 약점 파일에 추가 + 상세 설명**
   - `/Users/ihaneul/security-study/정보보안기사_Day2_약점정리.md`
   - 해당 섹션에 개념 추가
   - 체크리스트에 항목 추가

2. **초등학생도 이해할 수 있게 설명**
   - 전문 용어 최소화
   - 비유와 예시 활용
   - 왜 그런지 이유 설명

3. **시각화 필수**
   ```
   ┌─────────┐      ┌─────────┐
   │  개념A  │ ───→ │  개념B  │
   └─────────┘      └─────────┘
   ```
   - 흐름도, 비교표, 그림으로 표현
   - 관계와 차이점을 한눈에 파악 가능하게

### 설명 예시 형식

```
🎯 핵심 개념: [한 줄 요약]

📖 쉬운 설명:
[초등학생도 이해할 수 있는 비유]

🔄 흐름/구조:
[ASCII 그림 또는 표]

⚠️ 헷갈리는 포인트:
[자주 틀리는 부분 강조]
```
