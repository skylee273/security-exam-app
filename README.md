# 정보보안기사 학습앱

인터랙티브 정보보안기사 시험 대비 학습 도구

## 기능

- 학습 모드: 카드 형식으로 개념 학습
- 퀴즈 모드: 실전 문제 풀이
- 약점 모드: 틀린 문제 / 어려운 개념 집중 학습
- 진도 추적: 학습 완료 항목 체크
- 다크 모드: 눈 피로 감소
- 검색 / 필터: 원하는 내용 빠르게 찾기

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 배포

Vercel에 배포:

```bash
npx vercel
```

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
